## Project for intro to data science classes (Vistula University 2026/27)

### Installation & Setup

1. **Install uv** (if not already installed):
   [Command available here](https://docs.astral.sh/uv/#installation)

2. **Launch the app**:
   ```bash
   uv run main.py
   ```

That's it! `uv` will automatically install the required dependencies from `pyproject.toml` and run the application.
